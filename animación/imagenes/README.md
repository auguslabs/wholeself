# Imágenes para la animación

- **Fotos del equipo (slide 2):** La grid 3×3 usa imágenes desde `../public/rounded-decorative/` con nombres como `Alexandria-Rakes.webp`, `Amy-Melendrez.webp`, etc. Si en tu entorno las fotos están en otro sitio, copia aquí una carpeta `team/` con esos archivos `.webp` y actualiza en `index.html` la ruta a `imagenes/team/Nombre.webp`.

- **Home y servicios (slide 1):** Se usan `../public/banner-hero-section.webp` y `../public/banner-services.webp`. Si no existen en `public/`, puedes poner aquí `hero.webp` y `services.webp` y cambiar las rutas en `index.html`.
