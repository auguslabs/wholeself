## 🌐 Cómo publicar en GitHub Pages

### Opción 1: Usando la carpeta `presentations/` (Recomendado)

1. Ve a tu repositorio en GitHub
2. Click en **Settings** → **Pages**
3. En **Source**, selecciona:
   - **Branch:** `main` (o `master`)
   - **Folder:** `/presentations`
4. Click en **Save**
5. Espera unos minutos y tu sitio estará disponible en:
   ```
   https://[tu-usuario].github.io/[nombre-repo]/presentations/
   ```

### Opción 2: Usando la carpeta `/docs`

Si prefieres usar `/docs` (más común en GitHub Pages):

1. Renombra esta carpeta de `presentations/` a `docs/`
2. En GitHub Settings → Pages, selecciona `/docs` como carpeta
3. El sitio estará en: `https://[tu-usuario].github.io/[nombre-repo]/`

## 🔗 Compartir con el cliente

Una vez configurado GitHub Pages, comparte el link con tu cliente. El `index.html` mostrará una página bonita con enlaces a ambas presentaciones.

## 📝 Notas

- Los archivos HTML son estáticos y no requieren servidor
- Puedes actualizar las presentaciones y hacer push, GitHub Pages se actualizará automáticamente
- El link puede tardar unos minutos en estar disponible después de la primera configuración
